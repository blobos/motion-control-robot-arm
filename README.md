# Motion Control Popsicle-stick Robot Arm
Arduino
![image](https://user-images.githubusercontent.com/7018624/236264524-dbf8d309-f401-4df3-97fa-50235881d622.png)
#### Actuators:
- 4 Servo motors
- 1 Stepper motor
#### Sensors (Controls)
- MPU6050 gyroscope 
- Joystick
- 1-hand operation
